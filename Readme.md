# Portafolio Web - Adrian Gonzalez Valentin

Este repositorio contiene mi portafolio personal web.

---

##  Descripción del Proyecto

Este portafolio fue construido utilizando el framework CSS **Bootstrap 5**, aprovechando su sistema de cuadrícula y componentes responsivos para lograr un diseño "Mobile First". 

**Plantilla Utilizada**
El diseño base parte de la plantilla **"Resume" de Start Bootstrap**, la cual fue modificada estructuralmente y en contenido para adaptarse a mis necesidades personales y académicas.
**Enlace de descarga de la plantilla original:** [Start Bootstrap - Resume](https://startbootstrap.com/theme/resume)

**Estructura y Secciones del Portafolio**
*   **Menú de Navegación (SideNav):** Barra lateral fija que permite hacer scroll suave hacia las diferentes partes del documento.
*   **Sobre Mí:** Pantalla principal de inicio con mi nombre, titular profesional, un breve resumen de mi enfoque como desarrollador, correo de contacto y enlaces a mis redes (LinkedIn y GitHub).
*   **Proyectos:** Sección detallada con mis trabajos más destacados, incluyendo un Gestor de Tareas Interactivo, una aplicación con persistencia de datos en LocalStorage y una interfaz dinámica UI/UX.
*   **Formación:** Resumen de mi trayectoria académica actual como estudiante de programación.
*   **Habilidades:** Lista visual con iconos de FontAwesome que representan mi stack tecnológico (HTML5, CSS3, JavaScript, Bootstrap) y mi flujo de trabajo.
*   **Intereses:** Breve descripción de mis aficiones y metas dentro del mundo tecnológico.

---

##  Proceso de Creación paso a paso

**Paso 1: Selección y descarga del entorno base**
Descargué la plantilla "Resume" desde el sitio de Start Bootstrap para tener una base sólida con una navegación lateral fija, la cual me pareció ideal para un formato de currículum web.

**Paso 2: Limpieza y configuración del HTML**
Revisé el archivo `index.html`, actualicé las etiquetas `<meta>` para el SEO básico y modifiqué la etiqueta `<title>`. Posteriormente, limpié el contenido de relleno (Lorem Ipsum) que traía la plantilla por defecto.

**Paso 3: Personalización de la sección "Sobre Mí"**
Reemplacé la imagen del perfil por una fotografía personal (`adrian-foto.jpg`). Cambié los colores de los textos principales para resaltar mi nombre e integré mi correo de contacto real y una descripción personalizada sobre mis objetivos como desarrollador Frontend.

**Paso 4: Integración de Proyectos Propios**
Transformé la sección de "Experiencia" original de la plantilla en mi sección de "Proyectos Destacados". Aquí redacté la descripción de mis ejercicios prácticos (Actividad 3: Ejercicios 16, 17 y 18), explicando qué tecnologías usé en cada uno, como la manipulación del DOM y LocalStorage con JavaScript Vanilla.

**Paso 5: Adaptación de Habilidades e Iconos**
En la sección de "Habilidades", actualicé los iconos de FontAwesome para que reflejaran exactamente las tecnologías que manejo (agregando los iconos de JavaScript y Bootstrap). Modifiqué la lista de "Flujo de trabajo" para destacar el uso de Git, GitHub y diseño responsivo.

**Paso 6: Ajustes de interfaz y navegación**
Añadí un botón personalizado en el menú lateral (`Volver al Menú`) para mejorar la navegación del usuario en caso de que este portafolio esté anidado dentro de un proyecto más grande. Verificando finalmente que el scroll fluido funcionara correctamente.
